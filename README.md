# Pizza-Sales_Analysis
📌 Description
This project analyzes pizza sales data using SQL to uncover sales trends, customer preferences, and business performance. The analysis helps in understanding customer behavior and making data-driven business decisions.

🛠️ Tools & Technologies

- SQL
- MySQL
- Microsoft Excel

📂 Dataset

The dataset includes:

- 🆔 Order ID
- 🍕 Pizza Name
- 📋 Pizza Category
- 📏 Pizza Size
- 🔢 Quantity
- 📅 Order Date & Time
- 💰 Price

✨ Features

- 📈 Total Revenue Analysis
- 🛒 Total Orders Analysis
- 🏆 Best-Selling Pizzas
- 🍕 Popular Pizza Categories
- 📏 Sales by Pizza Size
- ⏰ Peak Order Hours Analysis
- 📊 Daily & Monthly Sales Trends

🔍 Key Insights

- ⭐ Identified top-performing pizzas.
- 👥 Analyzed customer ordering patterns.
- ⏰ Determined peak sales hours and busy days.
- 💵 Found the most profitable pizza categories and sizes.

🚀 How to Run

- 📥 Import the dataset into MySQL.
- 🗄️ Create the required database and tables.
- ▶️ Execute SQL queries.
- 📊 Review the analysis results and insights.

📁 Project Structure

- 📂 Dataset
- 📄 SQL Queries
- 📊 Analysis Results
- 📝 README File

 🎯 Conclusion
 
This project demonstrates how SQL can be used to transform raw sales data into meaningful business insights, helping organizations improve decision-making and understand customer preferences more effectively. 🍕📈🚀
